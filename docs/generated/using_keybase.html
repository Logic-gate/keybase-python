Using Keybase¶

The Keybase Class – Accessing Public User Data¶

The Keybase class allows you to find users in the Keybase directory and access their stored public keys. Public keys let you encrypt messages and files for a user; only the person holding the private key from the pair can decrypt a file encrypted with the public key.
